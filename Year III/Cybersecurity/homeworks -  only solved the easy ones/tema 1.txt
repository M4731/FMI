Exercitiul 6:
    malware = software creat pentru a deteriora sau pentru a obtine acces neautorizat la un computer/retea de computere.
    virus = cod care e capabil sa se multiplice pe sine si are de obicei rol negativ, cum ar fi coruperea unui pc.
    dropper = program care este destinat extractiei de fisiere malitioase.
    downloader = program capabil sa se instaleze singur si sa dea acces fisierelor straine la computer.
    trojan = program care opreste accesul exterior la pc si il controleaza singur.
    spyware = program care transmite informatiile computerului catre un alt hardware.
    riskware = un program care este riscant de folosit din cauza vulnerabilitatilor sale. Nu este neaparat malitios.
    ransomware = un software destinat sa blocheze accesul la o retea de calculatoare pana cand este platita o suma de bani.
    adware = software care downloadeaza si arata automat materiale de publicitate(pop-upuri sau flaguri) cand calculatorul este online.
    worm = software daunator care se copiaza pe sine in scopul de a se imparti la alte calculatoare.
    obfuscare = "uratirea" codului in scopul de a-l face ilizibil pentru a-i ascunde functionalitatea.
