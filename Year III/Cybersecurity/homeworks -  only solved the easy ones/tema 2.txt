Exercitiul 4:
    O functie f(n) este ne neglijabila daca exista o functie polinomiala g(n) mai mare decat aceasta ∀ n.

    1 - f(n) = 2, g(n) = 1 functie polinomiala  < f(n) = 2 oricare ar fi n => f este ne-neglijabila.
    2 - f(n) = 1/2000, g(n) = 1/2001 functie polinomiala < f(n) = 1/2000 oricare ar fi n => f este ne-neglijabila.
    3 - f(n) = 1/(n^2000), g(n) = 1/(n^2001) functie polinomiala < f(n) = 1/(n^2000) oricare ar fi n => f este ne-neglijabila.
    4 - f(n) = 1/(2^n) este neglijabila deoarece 2^n este exponentiala, deci nu exista o functie polinomiala care inversata sa fie mai mica decat 1/(2^n).
    5 - f(n) = f1(n) + f2(n), unde f1(n) si f2(n) sunt neglijabile  => f(n) este neglijabila (proprietate) <> suma a 2 functii neglijabile este tot neglijabila.
    6 - f(n) = f1(n) + f2(n), unde f1(n) este neglujabila si f2(n) este ne-neglijabila => f(n) ne-neglijabila. 
	exemplu : f(n) = 2<functie ne-neglijabila> + g(n) = 1/(2^n)<functie neglijabila> -> functie ne-neglijabila.
